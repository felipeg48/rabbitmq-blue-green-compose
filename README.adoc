= RabbitMQ Clusters - Blue/Green Deployments

*Blue*

[source,shell]
----
docker compose -f compose-blue.yaml up
----

- http://localhost:15672[^]

|===
| Port | Description
|15672 | Web Console with credentials `admin`/`admin`
|5672  | AMQP
|1936  | HAProxy Console with credentials `admin`/`admin`
|===
